## What's New / 新增功能

### EN

- **Ornn Core Skills** — Three built-in skills that teach your AI agent how to use Ornn:
  - `/ornn-search-and-run` — Find and execute any skill from the library in one command
  - `/ornn-build` — Describe what you need in plain language and AI generates a complete skill
  - `/ornn-upload` — Package and publish your skills so others can use them
- **Install with One Prompt** — Paste a single prompt into your AI agent to install all three core skills. No scripts, no manual setup.
- **4 Agent Platforms Supported** — Installation prompts for Claude Code, OpenAI Codex, Cursor, and Antigravity.
- **Go to NyxID** — Quick link to NyxID from your profile dropdown menu.
- **Version Roadmap** — A new page listing all released versions. Click any version to see what's new.
- **One-Click Copy** — Code blocks in documentation now have a copy button for easy clipboard access.
- **Admin Panel Fix** — Fixed a bug where the Admin Panel link disappeared after being logged in for a while.
- **Updated Documentation** — Rewritten quick start guide with real examples, use cases, and step-by-step walkthrough. NyxID integration reference moved to its own page.

### ZH

- **Ornn 核心技能** — 三个内置技能，教你的 AI Agent 如何使用 Ornn：
  - `/ornn-search-and-run` — 一条命令搜索并执行技能库中的任意技能
  - `/ornn-build` — 用自然语言描述需求，AI 生成完整技能
  - `/ornn-upload` — 打包并发布你的技能，让其他人可以使用
- **一键提示词安装** — 将一段提示词粘贴给你的 AI Agent 即可安装全部三个核心技能。无需脚本，无需手动操作。
- **支持 4 个 Agent 平台** — 为 Claude Code、OpenAI Codex、Cursor 和 Antigravity 提供安装提示词。
- **前往 NyxID** — 在个人头像下拉菜单中快速跳转到 NyxID。
- **版本路线图** — 新页面展示所有已发布版本，点击任意版本查看更新内容。
- **一键复制** — 文档中的代码块新增复制按钮，方便快速复制内容。
- **管理面板修复** — 修复了登录一段时间后管理面板入口消失的问题。
- **文档更新** — 重写快速入门指南，包含真实示例、使用场景和完整操作演示。NyxID 集成参考独立为单独页面。
